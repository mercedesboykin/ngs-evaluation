Slide 1 — Start Here

The PE firm pushing Run Payments
owns both companies.

This isn't speculation. Run Payments markets it directly: residual revenue flows to ISO partners on every transaction. That residual is paid by the merchant, not the PE firm.

Run Payments' Own Marketing

"Residual operations — automated complex payouts with configurable splits and audit trails."

This is one of Run Partner's core advertised features. It's the mechanism by which the PE firm earns money when their portfolio companies route payments through Run. There's nothing hidden about it — but it deserves to be on the table before NGS commits.

What this means in plain terms

The PE firm is not a neutral advisor on this decision. They're a vendor with a financial interest in NGS choosing Run Payments. The "synergy" argument inside the portfolio is real — for them. For NGS, every basis point of Run's processing fee includes the residual NGS is paying back to its own owner.

This isn't an indictment of the PE firm — it's how PE-owned payment ISOs work. But it changes how the recommendation should be evaluated: not as advice from your board, but as a vendor quote from a related party.

Why this matters for Tony's mandate

NGS is rebuilding after a $250K+ failed replatform. The board's confidence in this build depends on Tony making decisions that hold up to scrutiny. A payments decision driven by undisclosed portfolio economics — versus a vendor selection based on the merchant's actual needs — is the kind of choice that gets questioned later.

The path forward isn't to refuse Run Payments outright. It's to get the residual structure in writing, weigh it against what NGS gives up, and let the math speak.

This was the failure mode of the prior $250K replatform — payment friction in the buying flow. It cannot be the failure mode of this build.

Slide 4 — Be Fair

Where Run Payments
does have a real advantage.

This isn't a zero-sum comparison. Run has one legitimate edge worth taking seriously — and it's worth quantifying honestly rather than waving it away.

Level 2 / Level 3 data on commercial cards

Run's Virtual Terminal captures Level 2 and Level 3 transaction data, which can lower interchange on business, purchasing, and corporate cards by approximately 50–100 basis points. Shopify Payments doesn't currently pass full L2/L3 data.

For NGS, this matters most on transactions paid with corporate procurement cards — which is plausible for buyers like 24 Hour Fitness or LA Fitness placing large parts orders through their purchasing departments.

The honest framing

For roughly 10–20% of NGS's B2B volume that runs on corporate procurement cards, Run will save 50–100 bps on interchange. For the other 80–90% of volume (consumer credit, debit, ACH, store credit, deposits), Run costs more — either directly in processing fees or indirectly in lost B2B-native features.

The PE firm is selling the 10–20% advantage. They aren't talking about the 80–90% cost.

What we should do, not avoid

Quantify the L2/L3 advantage with real numbers.

Ask Brian or the prior processor for a sample month showing card mix — specifically the share of transactions on business, purchasing, and corporate cards. That single data point lets us calculate the actual L2/L3 dollar advantage to within $1,000/year. Then weigh it against the ~$18K/yr 3P fee leakage and the feature loss. If the math favors Run on a specific volume segment, we'd hear it out — but it needs to be measured, not assumed.

Slide 5 — For the PE Firm

Five questions that
deserve answers in writing.

Tony, Vince, and the board should be able to point to documented answers on each of these before NGS commits to Run Payments. These aren't gotcha questions — they're standard related-party due diligence.

Question 01

What residual structure does the PE firm earn on NGS's Run Payments transaction volume?

Run Payments markets residual operations as a core ISO feature. NGS is entitled to know what percentage of every transaction routes back to its own owner — and whether NGS is on the standard merchant rate card or carrying portfolio uplift.
Question 02

Is NGS receiving preferential pricing relative to non-portfolio Run merchants — and if so, by how much?

If the answer is "no," the synergy argument falls apart — NGS would be paying the same rates as any merchant off the street. If "yes," it should be documented in basis points, not adjectives.
Question 03

How will NGS handle vaulted-card charges on the 30–35% of orders that go through quoting?

Shopify B2B draft orders only allow vaulted cards that were saved via Shopify Payments on a previous B2B checkout. Run's customer vault lives outside Shopify B2B checkout. The workaround needs to be defined before commit.
Question 04

Has the PE firm modeled NGS's total cost including Shopify's 0.15% 3rd-party transaction fee?

On $12M in GMV, that's ~$18,000/year that Shopify collects from NGS for using a non-Shopify gateway on Plus. That fee disappears with Shopify Payments. Any cost comparison that omits it is incomplete.
Question 05

What is the contractual exit path if NGS later wants to move payments to Shopify Payments?

If Run is sold via a multi-year minimum, termination fee, or volume commitment, the cost of correcting the decision later belongs in today's math.
Slide 6 — The Call

Shopify Payments on Plus
is the right call for NGS.

Not because it's our product — because every requirement Tony, Brian, and DYODE confirmed on May 20 either requires it outright or works dramatically better with it.

The recommendation

Move forward with Shopify Payments as NGS's processor on Plus. The B2B-native feature set — vaulted cards on Company Location, net terms tied to catalog tier, deposits, ACH, store credit, PO capture at checkout — is what makes this replatform succeed where the last one failed.

The ~$18K/year of avoided 3rd-party transaction fees and the elimination of an undisclosed PE residual are real dollars. The L2/L3 advantage on corporate cards is real but limited — and worth measuring before assuming it changes the answer.

The next step is not to refuse Run Payments outright. It's to put the five questions in front of the PE firm and the board, and let the documented answers drive the decision.
